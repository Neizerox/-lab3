## Задача 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 1. List.map) для
последовательности

### Алгоритм решения

Функция getSeq рекурсивно читает строки с консоли: если строка пустая, завершает ввод; если строка содержит ровно одну цифру от 1 до 7, добавляет число в последовательность с помощью yield и рекурсивно вызывает себя через yield! для продолжения ввода; при ошибках выводит сообщение и также рекурсивно продолжает ввод. Функция toDayName сопоставляет числам от 1 до 7 соответствующие названия дней недели на русском языке. В функции main программа выводит приглашение, получает последовательность чисел через вызов getSeq, преобразует каждый элемент в название дня с помощью Seq.map toDayName, а затем выводит полученные названия с помощью Seq.iter.

## Задача 2

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для
последовательности


### Алгоритм решения

Функция countDigits определяет количество цифр в числе: она берет модуль числа с помощью abs, преобразует его в строку и возвращает длину этой строки. Функция countNumbersWithKDigits принимает последовательность чисел и параметр k, затем с помощью Seq.fold проходит по каждому элементу: для каждого числа вызывается countDigits, и если результат равен k, счетчик-аккумулятор увеличивается на единицу, начальное значение аккумулятора равно нулю. В функции main программа запрашивает у пользователя ввод чисел через пробел и значение k, затем с помощью обработки исключений проверяет корректность ввода. Строка с числами разбивается по пробелам с удалением пустых элементов и преобразуется в последовательность целых чисел. После вызова countNumbersWithKDigits результат выводится на экран. Если ввод содержит некорректные символы, программа перехватывает исключение FormatException и выводит сообщение об ошибке.

## Задача 3

### Текст задачи

Имена скольких подкаталогов в указанном каталоге начинаются с
заданного символа?

### Алгоритм решения

Сначала пользователь вводит путь к каталогу и символ для поиска, после чего проверяется, что символ был указан. Затем вызывается функция countDirsByFirstChar, которая с помощью Directory.GetDirectories получает массив всех подкаталогов в указанном пути, после чего через Seq.filter происходит фильтрация: для каждого каталога извлекается его имя с помощью Path.GetFileName, проверяется что имя не пустое и что его первый символ равен заданному. Отфильтрованная последовательность передается в Seq.length для подсчета количества элементов. Если в процессе выполнения возникает исключение (например, путь недоступен или отсутствуют права доступа), функция возвращает -1. В основной программе анализируется результат: если значение неотрицательное, выводится количество найденных каталогов, иначе сообщение об ошибке.
